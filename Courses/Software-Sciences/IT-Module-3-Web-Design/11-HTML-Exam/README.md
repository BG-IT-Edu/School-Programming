# Контролна работа: HTML

Тази контролна работа в два варианта е насочена към проверка и затвърждаване на знанията по **HTML**, като използва теоретични задачи с избираем отговор и практическа задача, свързана със създаването на HTML уеб страница.

## Методически указания
 - Изпитният вариант, който сме споделили, е **само примерен**.
