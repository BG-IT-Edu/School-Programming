# Графичен дизайн на уеб сайт

Урокът "**Графичен дизайн на уеб сайт**" запознава с основните етапи при визуалното проектиране на уеб сайт, елементите на графичния дизайн и създаването на графичен модел (mockup). 

## Съдържание на темата "Графичен дизайн на уеб сайт"
  - **Графичен модел (Mockup)**
    - **Основни етапи**
  - **Прототип (Prototype)**
  - **Eлементи** на **графичния дизайн**
    - **Цветова схема**
      - **​Пример:** Създаване на цветова схема
    - **Шрифтове (Типография)**
      - **​Пример:** Комбиниране на шрифтове
    - **Подредба** и **секции**
  - **Лого и бранд идентичност**
  - **​Пример:** Графичен модел (mockup) на уеб страница по избор

## Графичен дизайн на уеб сайт - упражнение
  - Задача за създаване на **графичен модел (mockup)** за **практическия проект**
  
## Методически указания
  - Урокът е задължителен за **всички** ученици
