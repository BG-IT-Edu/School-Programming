# Контролна работа: Проектиране и графичен дизайн на уеб сайт

Тази контролна работа в два варианта е насочена към проверка и затвърждаване на знанията по **Раздел 2.Проектиране и графичен дизайн на уеб сайт**, като използва теоретични задачи с избираем отговор и практическа задача, свързана със създаването на схематичен дизайн (wireframe).

## Методически указания
 - Изпитният вариант, който сме споделили, е **само примерен**.
