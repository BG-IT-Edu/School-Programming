# Основи на CSS

Урокът **„Основи на CSS“** запознава учениците с основните принципи на CSS и неговата роля при оформянето и стилизирането на уеб страници.

## Съдържание на темата „Основи на CSS“

- **CSS**
  - **Въведение** и **роля**
- **Основни компоненти** на **CSS**
  - **Селектори**, **свойства**, **коментари**
- **Добри практики** при **работа** със **CSS**
  - **Начини** за **добавяне** на **CSS**
- **Пример:** Стилизиране на уеб страница

## Основи на CSS - упражнение

- Въпроси с избираем отговор
- Задача за стилизиране на **HTML уеб страница**

## Методически указания

- Урокът е задължителен за **всички** ученици
