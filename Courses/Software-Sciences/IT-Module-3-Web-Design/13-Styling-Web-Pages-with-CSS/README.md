# Визуално оформяне на уеб страници със CSS

Урокът **„Визуално оформяне на уеб страници със CSS“** запознава учениците с използването на цветове, фон, шрифтове и подредба на елементите с цел създаване на по-четима, подредена и визуално завършена уеб страница.

## Съдържание на темата „Визуално оформяне на уеб страници със CSS“

- **Стилизиране** на **уеб съдържание**
- **Цветове** и **фон**
  - **Цветови стойности**
  - **Изображения** за **фон**
- **Шрифтове** и **текст**
  - **Външни шрифтове**
  - **Google Fonts**
- **Подредба** на **елементи**
  - **Grid система**
- **Пример:** Визуално оформление на уеб страница

## Визуално оформяне на уеб страници със CSS - упражнение

- Въпроси с избираем отговор
- Задача за стилизиране на **HTML уеб страница** за **практическия проект**

## Методически указания

- Урокът е задължителен за **всички** ученици
