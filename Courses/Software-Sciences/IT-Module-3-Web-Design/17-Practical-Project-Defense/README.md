# Защита на практически проект

В края на раздела учениците представят и защитават разработения **практически проект**.

Представянето включва **темата**, **целта** и **целевата аудитория** на уеб сайта, неговата **структура** и **основни страници**, използваните **HTML** и **CSS елементи**, както и основните **етапи** на **работа**. При **екипен проект** се представят **участниците**, **разпределението** на **задачите** и **приносът** на **всеки участник**.

Защитата завършва с **демонстрация** на **публикувания уеб сайт** и отговори на **въпроси**, свързани с неговото разработване.
