# Защита на практически проект

В края на раздела учениците представят и защитават разработения практически проект.

Представянето включва темата, целта, структурата и основните страници на уеб сайта, използваните HTML и CSS елементи и основните етапи на работа. При екипен проект се описват разпределението на задачите и приносът на всеки участник.

Защитата завършва с демонстрация на публикувания уеб сайт и отговори на въпроси, свързани с неговото разработване.
