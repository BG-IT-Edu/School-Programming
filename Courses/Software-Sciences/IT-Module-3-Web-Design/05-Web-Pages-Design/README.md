# Проектиране на уеб страници

Урокът "**Проектиране на уеб страници**" разглежда основните понятия и стъпките за проектиране и визуализиране на уеб страница чрез схематичен дизайн (wireframe).

## Съдържание на темата "Проектиране на уеб страници"
  - **Проектиране на уеб страници**
    - **Цел** и **основни етапи**
  - **Графичен интерфейс**
  - **Схематичен дизайн (Wireframe)**
    - **Основни елементи**
  - **Достъпност на уеб страници (Web Accessibility)**
  - **Типове уеб страници** 
  - **​Пример:** Схематичен дизайн на уеб страница по избор

## Проектиране на уеб страници - упражнение
  - Въпроси с избираем отговор
  - Задача за създаване на **схематичен дизайн (wireframe)** за **практическия проект**
  
## Методически указания
  - Урокът е задължителен за **всички** ученици
